# Random Quote Generator - Генератор случайных цитат

## Автор
**Орехов Дмитрий**

## Описание программы

**Random Quote Generator** — это приложение для генерации вдохновляющих цитат. Программа выбирает случайные цитаты из базы данных, сохраняет историю просмотров и позволяет фильтровать цитаты по автору и теме.

### Основные возможности:

- ✅ Генерация случайных цитат
- ✅ 15 предустановленных цитат разных авторов и тем
- ✅ Добавление новых цитат через интерфейс
- ✅ Удаление существующих цитат
- ✅ Сохранение истории всех сгенерированных цитат
- ✅ Фильтрация истории по автору и теме
- ✅ Просмотр полной цитаты по двойному клику
- ✅ Статистика: количество цитат, авторов, тем
- ✅ Сохранение данных в JSON файлы
- ✅ Валидация ввода при добавлении цитат

## Требования к системе

- Python 3.7 или выше
- Tkinter (входит в стандартную поставку Python)
